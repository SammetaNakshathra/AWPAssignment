This repository contains my AWP (Antenna and Wave Propagations) assignment implemented using MATLAB.
The project focuses on understanding core programming concepts through MATLAB scripts and live scripts, including data processing, logical operations, mathematical computations, and result visualization.
The assignment demonstrates how MATLAB can be used as a powerful tool for algorithm implementation, analysis, and simulation, making it suitable for academic and engineering applications.
